Checkpoint 3.2 Answers

1. Embedded Styles

A reason to use embedded styles is when you want to apply CSS to only one specific web page. This is helpful for quick styling without needing an external file. Embedded styles are placed inside a <style> element, which goes in the <head> section of the HTML document. 


  <head>
    <style>
      body {
        font-family: Arial, sans-serif;
        background-color: lightgray;
      }
    </style>
  </head>

2. External Styles

External styles are useful when the same CSS rules need to be applied to multiple web pages. This promotes consistency and easier updates across an entire website. External styles are stored in a separate CSS file, and the HTML page uses a <link> tag in the <head> to reference that file. 


  <head>
    <link rel="stylesheet" href="mystyles.css">
  </head>

3. Code to Use "mystyles.css"

To configure a web page to use an external style sheet named mystyles.css, add the following code inside the <head> section: 


  <head>
    <link rel="stylesheet" href="mystyles.css">
  </head>
